You are a senior SaaS architect and frontend engineer.

Your task is to build a **production-ready frontend for a SaaS Hotel Management System called "ROBO HMS"**.

The frontend must be built using:

* React (JavaScript, not TypeScript)
* Vite or Next.js (prefer modular structure)
* TailwindCSS for UI
* React Query / TanStack Query for API state
* React Router for navigation
* Axios for API calls
* Modular folder structure
* Fully responsive UI
* Clean enterprise design

The system must support **multi-tenant SaaS architecture**, where many hotels use the platform independently.

IMPORTANT RULE:
The platform has **two completely separate portals**:

1. HOTEL PORTAL (for hotel owners and staff)
2. ROBO ADMIN PORTAL (for SaaS platform management)

These portals must have **separate login systems and UI**.

Do NOT mix these two systems.

---

# PRODUCT OVERVIEW

ROBO HMS is a **complete Hotel Management System SaaS platform**.

Hotels can manage operations through:

1. Web dashboard
2. WhatsApp AI assistant
3. Guest booking website

Hotels subscribe to the platform and can **buy optional modules**.
